At the end of this lesson you should have a brief overview of:

  • JQuery
  • Bootstrap
  • Javascript and CSS

Bootstrap

Bootstrap is a CSS framework with predefined CSS code that enables you as a designer to easily style your page. And if responsiveness was your weakness,with Bootstrap,it's time to say goodbye.

Jquery

Just like Bootstrap, Jquery is a framework, but based on Javascript. It has predefined Javascript code that makes work a little bit easier for you as a developer.




Javascript

So Javascript is the mother to Jquery the way CSS is the mother to bootstrap. You're going to need to understand JS before you proceed to Jquery. As a fun fact, JS is the first programming language that more than 50% programmers familiarise themselves with on their programming journey
